# Cure Runner — Copy Lexicon

Living vocabulary and phrase reference for the project.

---

## 1. Preferred Vocabulary Families

### World and Nature
`planeta` · `tierra` · `interior` · `superficie` · `luz` · `aire` · `camino` · `paso` · `nota` · `voz` · `raíz`

These words anchor the game's world. They are concrete, visual, and carry symbolic weight without requiring explanation.

### Movement and Process
`subir` · `ascender` · `despertar` · `abrir` · `avanzar` · `volver` · `seguir` · `buscar` · `respirar`

Action verbs that imply ongoing process, not completed achievement.

### State and Observation
`hay` · `vuelve` · `comienza` · `se acerca` · `aún` · `poco a poco` · `algo`

Words that describe the state of the world, not the performance of the player.

### Warmth (used sparingly)
`juntos` · `sigamos` · `respira` · `arriba` · `toma`

Reserved for companionship moments. Never used for fake emotional closeness.

---

## 2. Discouraged Vocabulary Families

### Evaluation Words — NEVER USE
`increíble` · `genial` · `perfecto` · `excelente` · `bien hecho` · `bravo` · `campeón` · `crack`

These evaluate the person. The game evaluates nothing.

### Self-Help / Therapy Language — NEVER USE
`acepta` · `procesa` · `siente` · `valora` · `mereces` · `eres suficiente` · `confía en ti` · `déjalo ir` · `todo estará bien`

These import an adult therapeutic register that is condescending to a 12-year-old.

### Generic Motivation — NEVER USE
`no te rindas` · `tú puedes` · `cree en ti` · `todo es posible` · `sigue tus sueños` · `lo importante es participar` · `sigue intentando`

These are empty. A perceptive reader hears them as noise.

### Fear/Shame Language — NEVER USE
`cuidado` (as threat) · `no pierdas` · `inténtalo mejor` · `¿otra vez?` · `deberías`

These undermine autonomy and self-worth.

### Overexplaining Connectors — AVOID
`porque` · `para que` · `de esta forma` · `así podrás` · `recuerda que`

These make the game sound like a teacher explaining itself.

## 3. Mechanic Terminology

| Mechanic | Term | Why |
|----------|------|-----|
| Health/pulse | **Vida** | Universal, instant clarity, one word |
| Note progress toward reserve | **Reserva** | Self-describing: you're filling a reserve |
| Counter value | **0/100** | Bare number, no label needed beyond eyebrow |
| Collectibles (in-world) | **notas** | The word used in hints and guidance ("Notas llenan la reserva") |
| Helper creature | *(no name in UI)* | The shark appears, helps, and speaks. Naming it would reduce it. |
| Hazards | *(no label in UI)* | Visual only. No text labels on obstacles. |
| Level | **Nivel 1** | Standard, clear |
| World | **Wounded Planet** | English preserved as proper noun / title identity |

### Rules
- Mechanic labels are **never poetic**. "Vida" not "Energía vital." "Reserva" not "Legado."
- If adding a new counter or meter, the eyebrow must be **≤2 words** and **immediately self-evident.**
- Never rename a mechanic mid-session for variety. Consistency is clarity.

## 4. Emotional Vocabulary

### For World-Observation (allowed)
`despertar` · `despertado` · `luz` · `respira` · `vuelve` · `se acerca` · `aún` · `algo` · `comienza`

These describe the world's emotional state, which the player reads as an indirect mirror.

### For Player Emotions (forbidden as labels)
Never write: `te sientes` · `estás triste` · `estás feliz` · `no tengas miedo` · `eso duele`

The game does not name the player's emotions. The player is sovereign over their inner life.

### The Emotional Channel
Emotion travels through:
1. **Rhythm** — short sentences for urgency, longer for peace
2. **Image** — "La luz vuelve" instead of "las cosas mejoran"
3. **Silence** — no text is sometimes the most emotional choice
4. **Sound** — the audio cue system carries emotional weight parallel to copy

## 5. Symbolic Vocabulary

| Symbol | Meaning layer | How it appears in copy |
|--------|--------------|----------------------|
| **Luz** | Hope, healing, recovery | "La luz vuelve poco a poco." |
| **Interior** | Hidden depth, the self, what is below surface | "Abriendo el interior..." |
| **Despertar** | Growth, change, awareness | "Algo ha despertado." |
| **Respirar** | Safety, pause, being alive | "Respira." |
| **Camino** | Continuity, process, the ongoing journey | "El camino no se cierra." |
| **Planeta** | The world, shared responsibility, something larger than the self | "El planeta aún respira." |

### Rules
- Symbolic vocabulary earns its place through game mechanics. "Luz" works because brightness literally increases with progress.
- If a symbol has no mechanical anchor, it becomes fake profundity. Don't use it.
- Max 2 symbolic words per line. More than that obscures meaning.

## 6. Culturally Rooted Expression Guidelines

### Protected Expressions (do not remove or "neutralize")

| Expression | Origin | Context | Function |
|------------|--------|---------|----------|
| `"Ñó!!!"` | Cuban/Caribbean | Hit reaction | Family identity, authentic surprise, cultural truth |
| `"Ay!!"` | Pan-Hispanic | Hit reaction | Universal, warm, human |

### Integration Criteria for Future Expressions

Before adding any culturally specific expression, it must pass all four gates:

1. ✅ **Authentic** — Is this how real people actually speak?
2. ✅ **Legible** — Can someone unfamiliar with it understand from context?
3. ✅ **Meaningful** — Does it carry weight beyond decoration?
4. ✅ **Purposeful** — Is there a specific narrative/character/emotional reason for it?

If any gate fails → do not include.

### Density Rule
No more than 1 culturally specific expression per 30 seconds of gameplay. Density kills authenticity.

## 7. Tone Ladders by Context

### HUD Hints (during active gameplay)
```
Neutral:    "Toca para saltar."
Informative: "Notas llenan la reserva."
Urgent:     "Busca aire."
Confirming:  "Reserva activa."
```
All ≤5 words. No poetry. No evaluation. No exclamation marks.

### Guidance Lines (during recovery)
```
Gentle:     "Sigue subiendo."
Observed:   "Cada paso lo despierta un poco más."
Active:     "No te detengas."
Awe:        "La luz vuelve poco a poco."
```
≤7 words. Allowed one symbolic word that anchors to a physical action or visual state (e.g., using "luz" to mean progress because the screen literally brightens). Do not use abstract concepts without a visual anchor.

### Helper Creature (shark)
```
Breath:     "Respira."
Direction:  "Arriba."
Offering:   "Toma aire."
```
1–2 words. No explanation. The action (health restore) says everything else.

### Victory Panel
```
Title:      "Nota despertada"
Label:      "La cura comienza"
Body:       "Algo ha despertado."
Closing:    "La luz vuelve poco a poco."
```
Each line is a separate beat. Title is factual. Body is observed. Closing is atmospheric.

### Failure Panel
```
Title:      "Aún hay luz."
Body:       "El camino no se cierra."
CTA:        "Toca para volver."
```
Undefeated tone. Present tense. No reference to "next time" or "trying again."

### Continue Panel
```
Title:      "Respira."
Body:       "Cada paso despierta algo."
Closing:    "Sigamos."
```
Intimate. Brief. The shortest panel because the player is ready to move forward.
