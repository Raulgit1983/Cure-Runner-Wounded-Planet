# Cure Runner — Copy Review Rubric

Scoring system for evaluating any line or set of lines before they enter the game.

---

## How to Use

Rate each criterion from 0–4. A line must score **≥ 3.0 average** to ship. Any single criterion scored **0** is an automatic reject.

---

## Criteria

### 1. Clarity (weight: 1.5×)

> Can a 12-year-old on a phone understand this in under 2 seconds?

| Score | Meaning |
|-------|---------|
| 4 | Instantly clear; no ambiguity |
| 3 | Clear on first read with context |
| 2 | Requires a pause to parse |
| 1 | Requires re-reading |
| 0 | Confusing or misleading |

### 2. Dignity

> Does this respect the reader's intelligence and autonomy?

| Score | Meaning |
|-------|---------|
| 4 | Treats reader as equal; invites, never condescends |
| 3 | Respectful; no condescension but slightly generic |
| 2 | Slightly patronizing or overly directive |
| 1 | Evaluative or performance-judging |
| 0 | Condescending, praise-manipulative, or shaming |

### 3. Emotional Truth

> Does this feel honest, or does it feel performed?

| Score | Meaning |
|-------|---------|
| 4 | Rings true on the 200th reading |
| 3 | Honest; no false notes |
| 2 | Slightly generic but not dishonest |
| 1 | Feels like filler or routine |
| 0 | Self-help cliché, fake profundity, or manipulative |

### 4. Educational Value

> Does this create conditions for growth without sounding educational?

| Score | Meaning |
|-------|---------|
| 4 | Implicitly supports observation, agency, or resilience without the reader noticing |
| 3 | Neutral; doesn't harm and creates some space for reflection |
| 2 | Mildly didactic |
| 1 | Sounds like school material |
| 0 | Preachy, moralizing, or lecturing |

### 5. Mobile Readability

> Does this survive the physical constraints of a phone screen?

| Score | Meaning |
|-------|---------|
| 4 | ≤ word limit for its context; no line wrap; legible at target size |
| 3 | Within limits; fits on one line |
| 2 | Close to limit; may wrap on narrow screens |
| 1 | Too long; will definitely wrap |
| 0 | Requires scrolling or drastically exceeds word limit |

### 6. Conceptual Depth

> Does this carry meaning beneath the surface?

| Score | Meaning |
|-------|---------|
| 4 | Works on surface and resonates symbolically |
| 3 | Has one clear layer of meaning |
| 2 | Functional but flat |
| 1 | Generic; could be in any game |
| 0 | Empty or pretentious |

### 7. Non-Cliché Strength

> Is this original to this project, or is it recycled game writing?

| Score | Meaning |
|-------|---------|
| 4 | Could not exist in any other game; fully native to Cure Runner |
| 3 | Distinctive; fits this world specifically |
| 2 | Could appear in some other games |
| 1 | Common game writing pattern |
| 0 | Stock phrase; found in 100 other games |

### 8. Context Fit

> Does this belong in its specific gameplay moment?

| Score | Meaning |
|-------|---------|
| 4 | Perfect for the emotional and mechanical moment |
| 3 | Appropriate for the context |
| 2 | Acceptable but slightly off-register |
| 1 | Wrong tone for the moment |
| 0 | Actively harmful in this context |

### 9. Mission Alignment

> Does this serve the project's mission of accompaniment, education, and human growth?

| Score | Meaning |
|-------|---------|
| 4 | Directly advances the mission without visible effort |
| 3 | Aligned with the mission |
| 2 | Neutral to the mission |
| 1 | Slightly counter to the mission |
| 0 | Contradicts the mission (e.g., shames, manipulates, evaluates) |

---

## Scoring

- **Clarity is a Hard Gate:** If Clarity < 3, the line is rejected immediately, regardless of other scores.
- **Average Score:** Sum of all 9 criteria divided by 9.
- **Minimum to ship:** ≥ 3.0 average (with Clarity ≥ 3).
- **Auto-reject:** Any single criterion scored 0.
- **Excellence threshold:** ≥ 3.5 average.
