# Cure Runner — Copy Rules (Quick Reference)

Operational checklist for any agent or contributor writing copy for this project.
Read `copy-doctrine.md` for the full rationale.

---

## Before Writing Any Line

- [ ] **Read the doctrine** if you haven't this session
- [ ] **Identify the context code** (see taxonomy in doctrine §11)
- [ ] **Apply the decision hierarchy:** silence → clarity → warmth → poetry
- [ ] **Fast dev phase?** Use `[BRACKETS]` for functional literals (doctrine §13)

## The 10 Commandments

1. **Max 5 words** for any HUD hint. Max 3 words for any CTA.
2. **Never evaluate the player.** No "bien hecho," no "puedes hacerlo."
3. **Never explain emotions.** Create space, don't label.
4. **Acknowledge process, not outcome.** What happened in the *world*, not what the player *achieved*.
5. **If it wraps on a phone screen, it's too long.**
6. **Copy must survive 200 reads.** If it feels hollow on repetition, rewrite.
7. **Poetry only in paused contexts** (victory body, title, surface guidance). Never in HUD labels, CTAs, error text, or onboarding.
8. **Cultural expressions must pass all four gates:** authentic, legible, meaningful, purposeful.
9. **No generic inspiration.** No "cree en ti," no "tú puedes," no "no te rindas."
10. **Silence is an option.** If a moment works without text, use no text.

## Red Flags — Stop and Rewrite If You See

- An exclamation mark after praise
- The word "increíble," "genial," or "perfecto"
- A sentence telling the player how they feel
- A line longer than 35 characters
- A CTA that uses a command verb ("¡Inténtalo!")
- A failure line that references "trying harder" or "next time"
- A victory line that praises the player directly
- Poetry in a HUD label or error message
- A cultural expression used for decoration
- Any line you wouldn't be comfortable reading 200 times

## SDT Check (Self-Determination Theory)

Before committing a line, verify:

- **Autonomy:** Does this invite or command? ✅ Invite only.
- **Competence:** Does this acknowledge what happened, or judge performance? ✅ Observe only.
- **Relatedness:** Does this speak alongside the player, or above them? ✅ Alongside only.

## NVC Check (Nonviolent Communication)

- Is this an observation or an evaluation?
- Does this describe feelings or prescribe them?
- Does this acknowledge needs or impose them?
- Is this an invitation or a demand?

## Commit Checklist

- [ ] Line identified by context code
- [ ] Passes 200-read test
- [ ] ≤ word count for its context
- [ ] No anti-patterns from doctrine §8
- [ ] Passes SDT check
- [ ] Passes NVC check
- [ ] Reviewed by someone who can read it as a 12-year-old would
