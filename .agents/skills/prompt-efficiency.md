# Prompt Efficiency

## Purpose
Keep AI collaboration short, high-signal, and reviewable.

## When To Use
- Before asking for the next implementation slice
- While refining prompts, plans, or architecture requests

## Required Checks
- Ask for one concrete slice at a time.
- Name the exact files, constraints, and validation target.
- Prefer diffs, decisions, and commands over long rationale.
- Reuse the project constitution and skills instead of restating them.

## Output Format
- Objective
- Scope
- Files
- Validation

## Rollback / Failure Notes
If a prompt invites a wide refactor, rewrite it to target one bounded improvement.
